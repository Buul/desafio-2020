# Front-End desafio RBOOH

Projeto baseado em documentação enviada pelo aplicador do desafio.

## Rodando o Projeto

#### Requisitos

- NodeJS 12.10.0 +
- NPM 6.10.3 +

Pronto agora basta instalar as dependências com `npm install` e rodar o projeto `npm start`.
